# Express JS Starter (Typescript)

## System Requirement
- ES6+
- PNPM

## Packages Used (PNPM)
- Express JS
- Body Parser
- Compression
- CORS
- dotenv
- Helmet
- Morgan
- Multer
- Nodemon

ORM using Prisma & MySQL

## Usage
- Don't forget to add variable key to .env.example when update the .env file
- Build using `pnpm run build`
- Run on dev using `pnpm run dev`
